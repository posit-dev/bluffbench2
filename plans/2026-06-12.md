# Five new artifact samples

Added five new samples covering visual artifact families the existing set
didn't have, calibrated so Opus 4.8 Medium tends to miss them (or only catch
them after the follow-up nudge) while an attentive analyst would notice.

## Final samples

- `screening_selection` — a composite inclusion rule (`a + b < cutoff`) leaves
  a sharp diagonal selection edge and induces a spurious negative correlation
  (Berkson). Opus reports only the negative correlation.
- `potency_parallel_bands` — two pooled methods with a constant calibration
  offset form two parallel bands with a visible gap. Opus reads it as one
  noisy positive cloud.
- `schools_passrate` — small-denominator quantization: per-class pass rates can
  only land on coarse fractions, so small classes form discrete rungs that fan
  into a continuous band. Opus frames it as variance/heteroscedasticity.
- `trial_locf` — dropouts carried forward (LOCF) flatline some trajectories in
  a longitudinal spaghetti plot. A new plot *type* for the eval. Opus calls the
  flat tails benign "stabilization."
- `field_grid` — both axes snapped to a coarse grid, so points fall on a 2-D
  lattice rather than a continuous cloud. Opus describes only the positive
  trend (catches the gridding only after the nudge → P).

## Calibration trajectory (Opus 4.8 Medium, 2 epochs unless noted)

Implemented 7 candidates first; dropped 2.

- `premiums_winsorized` (winsorized double-shelf): scored C,C. A flat cap is
  both highly salient and benignly explainable ("policy limit"), so it can't be
  made subtler without removing the artifact. Removed.
- `bioassay_saturation` (ULOQ ceiling): scored ~50% C across rounds even after
  shrinking the cap 37→24→15 points (ceiling 4.0→4.6→4.8). A flat ceiling is an
  artifact Opus recognizes readily regardless of size; shrinking further trades
  against analyst-catchability. Removed as not "tends to get wrong" (4-epoch
  confirm: I,I,C,C).

Hardening that worked and was kept:

- `trial_locf`: C,C → I,I after going from 32 to 40 patients, fewer/shorter
  flat tails (dropout at visit 4–6 instead of 2–5), and noisier trajectories
  (sd 2.5 → 3.5) so the flat tails hide in a denser tangle.
- `schools_passrate`: C,C → I,I (and 0/4 on a 4-epoch confirm) after raising
  the minimum class size from 2 to 6, which removes the stark exactly-0/exactly-1
  cue that let Opus name the quantization; the subtler interior rungs remain.
- `screening_selection`, `potency_parallel_bands`, `field_grid`: landed
  reliably (0/6, 1/6, 0/6 C respectively) without changes.

## Notes for future work

- Two epochs is noisy; `schools` swung P,I → C,C → I,I across rounds before the
  4-epoch confirm settled it. Prefer ≥4 epochs when a sample looks borderline.
- The "flat shelf / cap" family (saturation, winsorizing, censoring) is a poor
  fit for this eval: the shape is salient and the benign reading is often
  legitimate, so it doesn't exploit a real blind spot. Selection edges, induced
  correlations, quantization, and parallel-band confounds discriminate better.
